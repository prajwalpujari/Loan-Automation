# Loan Automation

## Description
Loan Automation is a Python-based project that automates the loan processing workflow.  
It helps in evaluating loan applications by analyzing input data and applying predefined rules or logic to determine loan eligibility.

## Features
- Automated loan approval process
- Input data validation
- Rule-based decision making
- Simple and easy-to-understand implementation

## Technologies Used
- Python
- Pandas
- NumPy

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/prajwalpujari/Loan-Automation.git
2. Install required dependencies:
3. ```bash
   pip install -r requirements.txt
4. Run the project:
5. ```bash
   python main.py
